## Installation avant de lancer le projet

Il faut installer Node.JS sur votre ordinateur pour faire tourner le projet.

## Commandes

Via un terminal, dans le dossier du projet :

### `npm run dev` ou `npm start`

Lance le projet en mode developpement.
Toutes modifications dans les fichiers rechargera la page.<br>
